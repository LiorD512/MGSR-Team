Feature Specification

Men Platform Only

Inner chat room for all agents — like a team WhatsApp group.

💬 Core Chat

  • Real-time group chat visible to all team members
  • Messages stored in Firestore (chatRoom collection)
  • Sender name displayed in user's chosen language (EN/HE)
  • Timestamp on every message
  • Auto-scroll to latest message on open

🔔 Targeted Push Notifications

  • Choose a recipient user before sending
  • Only selected user gets push notification
  • Push click → opens chat room
  • Scrolls to & highlights the referenced message
  • Highlight fades after 3 seconds

⚽ @Player Mentions

  • Type @ to trigger player search dropdown
  • Search from men players database (Firestore)
  • Selected player renders as clickable chip/link
  • Click player name → navigate to PlayerInfo screen
  • Shows player position & team in dropdown

📱 Platform Support

  • Android: New route in NavGraph (ChatRoomScreen)
  • Web: New page route /chat-room
  • Men platform only (no women/youth)
  • RTL support for Hebrew users
  • Navigation from sidebar/bottom nav
