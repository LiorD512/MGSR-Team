משימות שחקן — הרעיון

משימה שנוצרת מעמוד שחקן מקבלת אוטומטית קונטקסט לשחקן. המשימה מופיעה גם אצל הסוכן שיצר אותה וגם בעמוד השחקן.

  • קישור כפול: משימה מקושרת לסוכן (מי יצר/אחראי) וגם לשחקן
  • טמפלטים: במקום טקסט חופשי — בחירה מטמפלטים ספציפיים לשחקנים (לדוגמה: "לדבר בחודש מרץ לבדוק סטטוס")
  • יעילות: פחות הקלדה, יותר עקביות, חיפוש וסינון קלים יותר
  • נראות: בעמוד השחקן — כל המשימות הקשורות אליו; בעמוד המשימות — עם תגית/קישור לשחקן

גישה משני מקומות

מעמוד המשימות — לחיצה על תגית השחקן → מעבר לעמוד השחקן. מעמוד השחקן — רואים את כל המשימות הקשורות

  playerId?: string      // חדש — קישור לשחקן

  templateId?: string    // אופציונלי — איזה טמפלט נבחר
}
