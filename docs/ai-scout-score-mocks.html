AI Scout Score — סקירת הפיצ'ר

ציון אוטומטי (0-100) לכל שחקן במאגר, בשילוב המלצת ליגה מדויקת. הציון לוקח בחשבון סטטיסטיקות FBRef, גיל, שווי שוק, רמת הליגה, ניסיון, ועוד.

🔄 השוואת אפשרויות מיקום

אפשרות מיקום פירוט מלא סקירה מהירה ליגות מותאמות השוואה מורכבות פיתוח
A: עמוד שחקן Hero + פאנל ייעודי בעמוד ✓ מלא ◐ דורש כניסה לעמוד ✓ מפורט ✗ שחקן בודד בינונית
B: רשימה + ציון עמודת ציון ברשימת שחקנים ✗ רק מספר ✓ מיידי ◐ בדג׳ מינימלי ✓ קל מאוד נמוכה
C: כרטיסיות כרטיסיית שחקן מורחבת ◐ מקוצר ✓ מיידי ✓ עם בדג׳ים ✓ grid להשוואה בינונית
D: טאב ייעודי טאב חדש "דו״ח סקאוט" ✓ מקסימלי ✗ דורש מעבר טאב ✓ מפורט מאוד ✗ שחקן בודד גבוהה
E: Hover/Tooltip ריחוף מעל שחקן ✗ מינימלי ✓ מהיר מאוד ◐ בדג׳ אחד ✓ מהיר נמוכה
💡

ההמלצה שלי: שילוב A + B

ציון קומפקטי ברשימת שחקנים (אפשרות B) לסריקה מהירה + פאנל מפורט בעמוד שחקן (אפשרות A) עם פירוט ציון + התאמת ליגות. זה נותן את שני העולמות — Quick scan ו-Deep dive — בלי להכביד על ה-UX. אפשר להוסיף גם Tooltip (E) כשלב ביניים.

🏗️ מבנה הציון המוצע

נוסחת הציון (0-100)

1. ביצוע סטטיסטי 40%
Per-90 stats מותאם לעמדה (כבר קיים ב-similarity.py). חלוץ = שערים, xG, בעיטות. קשר = מסירות מפתח, תיקולים, carries. בלם = תיקולים, יירוטים, אווירי. שוערים = ללא ציון סטטיסטי (ציון חלקי).
2. רמת ליגה 20%
דירוג ליגה 1-5 (כבר יש 40 ליגות במאגר). Premier League=100, Championship=70, ליגת העל=55, ליגה לאומית=35. שחקן שמציג ביצועים ב-Tier 1 מקבל בונוס.
3. פרופיל 20%
גיל (21-26 = מקסימום), שווי שוק, גובה מותאם לעמדה, חוזה (יותר זמן = יותר יציבות), רגל (דו-רגלי = בונוס).
4. עקביות ומשחקיות 10%
דקות 90 (יותר דקות = אמינות גבוהה יותר), כרטיסים.
5. נגישות 10%
שווי שוק ביחס לליגות יעד (ישראל: עד €2M = נגיש), חוזה (שנה אחרונה = בונוס), מצב השאלה, Free Agent.

⚠️ מגבלות ודיוק

מה הציון יודע לעשות טוב

  • סיווג שחקנים לרמת ליגה מתאימה (T1-T5)
  • זיהוי שחקנים עם ביצועים חריגים ביחס לליגתם
  • השוואה בין שחקנים באותה עמדה
  • חישוב מקומי מהיר — לא דורש API חיצוני
  • מבוסס על 19,749 שחקנים = מדגם רחב
  • מותאם לעמדה — כל פוזיציה נמדדת אחרת

מה הציון לא יודע / מגבלות

  • שוערים — אין נתוני saves/clean sheets, ציון חלקי בלבד
  • מהירות / אתלטיקה — אין נתוני GPS, רק proxy עקיף
  • שחקנים עם מעט דקות — ציון פחות אמין (פחות מ-6 משחקי 90)
  • ליגות לא במאגר — יקבלו ציון "מינימלי" ללא FBref
  • טלנט צעיר בליגה קטנה — עלול לקבל ציון נמוך מדי
  • אין ניתוח וידאו / Eye test — סטטיסטיקה בלבד
